# SSMA-Bot Telegram

## Objetivo

Desenvolver um bot para Telegram capaz de controlar os vencimentos de documentos relacionados à Segurança e Saúde no Trabalho.

## Escopo da versão 1.0

- Cadastro de colaboradores
- Cadastro de documentos
- Controle de vencimentos
- Consulta por matrícula
- Consulta por nome
- Relatório de pendências

## Tecnologias

- Python
- Telegram Bot API
- SQLite
- Git
- GitHub

## Futuras melhorias

- Dashboard Web
- API REST
